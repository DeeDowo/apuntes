HTML debug

La parte final de este apunte. Una vez finalizado nuestro documento HTML, lo recomendable es comprobar que este sea correcto. Una de las formas más fáciles es utilizando una página validadora, sin embargo, hay que tener presente que esta clase de herramientas sirven principalmente para identificar errores en la sintaxis del documento.

Un validador no puede determinar si el contenido es semánticamente adecuado o accesible; solo verifica si las etiquetas están bien anidadas, si los atributos son válidos, o si hay errores de cierre o estructura.

Una herramienta comúnmente usada para este fin es el validador oficial del W3C Basta con pegar el código o subir el archivo para recibir un reporte detallado.

Además de un validador, también es buena práctica usar el inspector de elementos del navegador (normalmente con clic derecho > "Inspeccionar") para verificar visualmente la estructura del documento y comprobar cómo el navegador interpreta cada sección.
