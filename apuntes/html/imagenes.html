Imágenes

Para insertar imágenes en un documento necesitamos el elemento <img> en conjunto con al menos los siguientes atributos:

  • src="URL": indica dónde se encuentra el archivo (recomendable el uso de rutas relativas).
  • alt="descripción de la imagen": proporciona una descripción breve de la imagen.

El atributo alt es especialmente útil cuando:

  • La imagen no puede cargarse (por conexión o preferencia).
  • Se utiliza un lector de pantalla (accesibilidad).
  • Se desea mejorar el SEO de una página.

Algunas buenas prácticas con imágenes

  • Usar nombres claros y breves para los archivos.
  • Definir dimensiones con los atributos width y height (solo para precargar el espacio. Idealmente, el tamaño intrínseco de las imágenes debería ser el mostrado).
  • El atributo alt solo debería usarse cuando previamente no se ha explicado la imagen.

Imágenes HTML vs CSS

Si bien podemos realizar las mismas acciones visuales con imágenes agregadas por medio de HTML o CSS, estas no son iguales.

Origen Función
HTML Significativas para el contenido, afectan a la comprensión o el propósito del documento mismo.
CSS Uso decorativo, no afectan al documento más que de manera visual.

<figure> y <figcaption>

Estos elementos trabajan en conjunto. Envuelven y describen cualquier contenido que pueda beneficiarse de una descripción (no solo imágenes).

  • <figure> ... </figure> — Agrupa.
  • <figcaption> ... </figcaption> — Describe el contenido agrupado.

Licencias de imágenes

No todas las imágenes en internet están disponibles para su libre uso. Es fundamental siempre respetar los derechos de autor.

Licencia Descripción
All rights reserved El creador debe dar permiso explícito para cualquier uso. Al momento de crear algo, por defecto, el autor posee todos los derechos.
Permissive licenses Permiten el uso libre, pero pueden requerir algunas condiciones, como el no uso comercial o el tener que dar créditos al autor o que el contenido generado a partir de este debe ser compartido igualmente por mencionar algunos ejemplos.
Public domain Cualquiera puede hacer lo que desee con el contenido, sin restricciones ni necesidad de permisos.

Una buena práctica es siempre asumir que todo en internet tiene algún tipo de copyright, salvo que se indique explícitamente lo contrario.
