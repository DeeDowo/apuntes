Imágenes

Para insertar imágenes en un documento, necesitamos el elemento <img> en conjunto con al menos los siguientes atributos:

  • src="URL" (Indica donde se encuentra el archivo, recomendable el uso de rutas relativas)
  • alt="descripción de la imagen" (Proporciona una descripción breve de la imagen)

El atributo "alt" es especialmente util cuando:

  • La imagen no pueda cargarse (por conexión o preferencia)
  • Para lectores de pantalla (accesibilidad)
  • Para mejorar el seo de una página

Algunas buenas practicas con imágenes

  • Nombres claros y breves para los archivos
  • Definir dimensiones con los atributos "width" y "height" (solo para precargar el espacio. Idealmente el tamaño intrínseco de las imágenes debería ser el mostrado)
  • El atributo "alt" solo debería usarse cuando previamente no se ha explicado la imagen

Imágenes HTML vs CSS

Si bien podemos realizar las mismas acciones visuales con imágenes agregadas por medio de HTML o por medio de CSS estas no son iguales.

Imágenes HTML vs Imágenes CSS
Origen Función
HTML Significativas para el contenido, afectan a la comprensión o el propósito del documento mismo,
CSS Uso decorativo, no afectan al documento más que de manera visual.

Figure y figcaption

Estos elementos trabajan en conjunto, envuelven y describen cualquier contenido que pueda beneficiarse de una descripción (no solo imágenes).

  • <figure> ... </figure> (Agrupa)
  • <figcaption> ... </figcaption> (Describe el contenido agrupado)

Licencias de imágenes

No todas las imágenes en internet están disponibles para su libre uso, es fundamental siempre respetar los derechos de autor.

Tipos de licencias
Licencia Descripción
All rights reserved (todos los derechos reservados) El creador debe dar permiso explicito para cualquier uso. Al momento de crear algo, por defecto, el creador tiene todos los derechos.
Permissive licenses (licencias permisivas) Permite el uso libre, pero puede requerir algunas condiciones, por ejemplo:
  • Dar créditos al autor
  • No uso comercial
  • Cualquier creación debe ser compartida
Public domain (dominio publico) Cualquiera puede hacer lo que desee con dicho contenido sin restricciones ni necesidad de permisos;

Una buena practica es siempre asumir que todo en internet tiene algún tipo de copyright, salvo que lo contrario se indique explícitamente.
