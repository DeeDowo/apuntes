Estructura general

Existe una estructura que debe tener cualquier documento HTML para poder ser interpretado correctamente.

<!DOCTYPE html>
Declara el tipo de documento y asegura que el navegador logre interpretar este de forma correcta.
<html>...</html>
Es el elemento raíz que envuelve todo el contenido del documento.
<head>...</head>
Incluye todo lo que no es contenido visible (metadatos, enlaces a recursos o configuraciones de comportamientos).
<body>...</body>
Incluye todo el contenido visible en la página. este tiene a la vez su propia jerarquía.
