Introducción a HTML

HyperText Markup Language es el lenguaje de marcado estándar usado para:

  • Estructurar el contenido de una página web (texto, imágenes, enlaces, etc.).
  • Definir la semántica de cada elemento (títulos, párrafos, listas, etc.) en una página web.
  • Ser la base fundamental sobre la que se añaden estilos (CSS) y funcionalidades (JavaScript).

En otras palabras, se puede decir que su finalidad es dar estructura y significado al contenido de una página web.
