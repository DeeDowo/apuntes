Formularios

Los formularios permiten recopilar información del usuario mediante diversos controles de entrada. su estructura semántica se conforma principalmente por:

Elemento Descripción
<form> ... </form> El elemento principal que se encarga de contener todos los elementos de un formulario.
<fieldset> ... </fieldset> y <legend> ... </legend> "Fieldset" se encarga de agrupar elementos relacionado dentro de un formulario, este se usa en conjunto con "legend", el cual debe ser el primer elemento anidado y contiene un breve titulo para este grupo.
<label> ... </label> Define una etiqueta para un control de entrada de un formulario, mejorando la usabilidad y accesibilidad. En otras palabras, indica el nombre del campo de entrada. Este elemento se asocia con un "input".
  • Asociación implícita (menos flexible, cuando el control de entrada se anida dentro del elemento label)
  • Asociación explicita (más flexible, se logra mediante el atributo "for" en label, que debe coincidir con el "ID" del control correspondiente)

tipos de entrada

Existen dos tipos de entrada en los formularios

Input

Permite diferentes tipos datos mediante el atributo "type", cada tipo proporciona un comportamiento predefinido. A su vez existen más tributos que pueden modificar aun más el comportamiento de estos. Algunos de los tipos más comunes son:

  • text
  • email
  • password
  • number
  • checkbox
  • radio
  • file
  • date
  • color
  • range
  • tel
  • url
  • search
  • hidden

Más sobre checkbox y radio

Estos 2 tipos de "input" en particular, sirven para marcar opciones. En el caso de "checkbox", se pueden elegir una o más opciones. Mientras que en el caso de "radio", solo se puede marcar una de las opciones , generalmente, las diferentes opciones se envuelven en un fieldset para una mayor claridad.

Checkbox

Los tres atributos esenciales para usar "checkbox" son:

  • type (Indica que le "input" será un "checkbox")
  • name (Es el atributo que se encarga de relacionar a los diferentes checkbox)
  • value (El valor de la casilla)
Ejemplo checkbox

Radio

Los tres atributos esenciales para usar "radio" son:

  • type (Indica que le "input" será un "radio")
  • name (Es el atributo que se encarga de relacionar a los diferentes "radio")
  • value (El valor del "radio")
