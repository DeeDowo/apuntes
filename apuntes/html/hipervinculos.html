Links

Un hipervínculo, link o enlace, permite conectar un documento con otro, vincular partes especificas dentro de documentos o incluso crear aplicaciones web. Los enlaces son fundamentales en la estructura de la web.

Un link se define mediante el elemento "anchor" y el atributo "href" que especifica el destino del enlace, es decir, la URL.

  • <a href="www.ejemplo.com"> Ejemplo </a>

No solo los fragmentos de texto pueden ser usado como links, cualquier elemento que esté envuelvo por el elemento "anchor" puede ser un link. Como bloques de texto, botones, imágenes, etc.

  • <a href="..."><span> ... </span></a>

Links a elementos de un documento

Es posible enlazar una parte especifica dentro de un documento html mediante un link, para hacer esto, el destino debe tener un "ID". Si el elemento a enlazar es del mismo archivo, solo se indica el "ID" con el prefijo '#'. Cuando el elemento no se encuentra en el mismo archivo, se debe indicar la URL y el "ID" con su prefijo '#'.

  • <a href="#id"> Ejemplo </a> (Enlace a un elemento)
  • <a href="/carpeta/index.html#id"> Ejemplo </a> (Enlace a un elemento fuera del documento actual)

Uniform Resource Locator

La URL indica la dirección de algo, ya sea un elemento, documento, archivo, pagina web, etc. Existen dos tipos, URL relativas y URL absolutas.

URL relativa

Aquella que se indica tomando un punto de referencia, si la referencia cambia, la ruta cambia.

  • "/carpeta/carpeta2/image.png"

URL absoluta

Aquella que se sin importar desde donde se referencie, no cambia.

  • "https://www.ejemplo.com/pagina.html"

Enlaces a recursos

Cuando un link lleva a algo que no es una pagina, por ejemplo, un pdf,imagen o archivo descargable, es recomendable:

  • Indicar el tipo de recurso (Formato y tamaño)
  • Explicar si será una descarga o una previsualización

Para una descarga se utiliza el atributo dowload (boolean), se iniciará la descarga de ser posible y si le asignamos un valor al atributo, el archivo se descargará con este nombre.

para abrir algo en una pestaña nueva, se utiliza el atributo "target" con el valor "_blank". Por defecto todos los links se abren en la misma página.

Algunas buenas practicas para el uso de links

  • Claridad textual (palabras cortas)
  • Claridad visual (debe entenderse a simple vista que es un link)
  • No repetir nombres en la medida de lo posible

Especialmente suele ser una buena practica indicar explícitamente o visualmente qué se realizará, ya sea una descarga, abrir en una nueva pestaña o cualquier comportamiento posible (la forma mas común es por medio de iconos).
