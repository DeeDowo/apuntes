Sintaxis básica de HTML

HTML posee reglas internas sobre el cómo utilizar las diferentes etiquetas disponibles, si bien un uso incorrecto de estas no ocasiona un error critico, es una buena practica en general el seguir dichas reglas

Elementos disponibles en HTML (etiquetas)

Elementos normales
son aquellos que que tienen tanto una etiqueta de inicio como una etiqueta de cierre.
Elementos vacíos
Son aquellos que solo poseen una etiqueta, estos no pueden tener contenido anidado.

Atributos HTML

Los elementos descritos anteriormente pueden contener atributos, es decir, características adicionales. Los atributos se insertan dentro de las etiquetas y estos se dividen en dos tipos:

Atributos normales
Los atributos normales poseen un nombre clave y un valor.
Atributos booleanos
Los atributos booleanos, a diferencia de los atributos normales, no requieren un valor necesariamente ya que su mera presencia indica un valor "true" en estos, es decir, aplican sus características por el simple hecho de encontrarse en el elemento.

Para que el valor de un atributo sea expresado e interpretado de forma correcta siempre, este debe estar entre comillas dobles ("valor_atributo") o simples ('valor_atributo'), pero no una combinación de ambas.
