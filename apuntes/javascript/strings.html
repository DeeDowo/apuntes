Strings

Los "strings" son cadenas de texto, este es un tipo de dato en JavaScript. Para inicializar una variable con un "string", hay cuatro formas principalmente, estas son:

Comillas simples
El texto a asignar se envuelve en comillas simples.
Comillas dobles
El texto a asignar se envuelve en comillas dobles.
Backticks
A diferencia de las comillas simples o dobles, los strings asignados usando "backticks", son considerados como un tipo especial, llamado "template literal". En esencia estos son iguales que los strings normales, pero tienen ciertas propiedades especiales:
  • Se pueden insertar variables o código JavaScript en ellos por medio de "${}".
  • Los "template literals" se pueden declarar en múltiples líneas.
Otra variable
Se asigna una variable que contenga algún string a otra.

Concatenación

A la unión de diferentes "strings" se le llama concatenación. Existen dos formas principalmente:

Operador +
Se pueden unir diferentes strings utilizando el operador "+", estos strings pueden ser literales o estar almacenados en una variable.
Template literals
La forma más común de hacer una concatenación es utilizando "template literals" en conjunto con "${}" que es la notación para poder agregar variables, operaciones o código JavaScript dentro de un "template literal".

Comillas

La forma más sencilla para poder agregar comillas en un string es usando uno diferente al que estemos usando para inicializar el string u otra opción es el usar caracteres escapados ("\'" o "\"").

Numbers vs strings

Al intentar hacer una operación entre un string y un número, JavaScript realiza una conversión implícita y hace que el número se convierta en un string, entonces la operación que se termina realizando es una concatenación.

Existen funciones para hacer estas conversiones entre tipos de forma explícita, algunas de ellas:

Number()
Convierte algo a un número.
String()
Convierte algo a un texto.

Métodos para strings

Al declarar una variable de algún tipo, en realidad estamos declarando una instancia de un objeto de dicho tipo, de esta forma es que tenemos acceso a diferentes métodos para diferentes tipos de variables. Por ejemplo, para un string algunos métodos y propiedades comunes son:

Length
Para poder saber el tamaño de string, podemos usar la propiedad "length". Para poder aplicar esta propiedad la notación es "string.length", esto nos devolverá la cantidad de caracteres que posea un string.
Include()
Para encontrar si algún substring se encuentra dentro de un string. Esto se puede realizar usando el método "include()", al cual como parámetro se le asigna el string a buscar, este método devuelve true o false dependiendo de si encuentra o no el substring
StartsWith()
Similar a "include()". El método "startsWith()" sirve para encontrar un substring específicamente al inicio de un string.
EndsWith()
Similar a "include()". El método "endsWith()" sirve para encontrar un substring específicamente al final de un string.
IndexOf()
Para encontrar la posición de un substring utilizando podemos utilizar el método "indexOf()". Este método toma dos parámetros, el primero es el substring que queremos encontrar y un segundo que especifica el punto de partida en el string para la búsqueda (índice), sin embargo, este segundo parámetro es opcional. Cuando "indexOf()" encuentra el substring, devuelve el índice de en dónde se encuentra el inicio de este, cuando no lo encuentra devuelve "-1".
Slice()
Para "extraer" un fragmento de un string, utilizamos el método "slice()", Este toma dos parámetros, el índice de inicio y el índice del fin de la parte del string que queremos extraer. El punto de inicio es inclusivo, esto significa que se incluye en la extracción mientras que el punto final el exclusivo, es decir, este no se incluye en el string extraído. Similar a "indexOf()", el segundo parámetro no es obligatorio, cuando este no se agrega, la extracción es desde el índice de inicio hasta el final del string.
ToLowerCase()
Para convertir todos los caracteres de un string a minúsculas, utilizamos "toLowerCase()", este método no toma parámetros. Una práctica común es el normalizar texto utilizando "toLowerCase()".
ToUpperCase()
Para convertir todos los caracteres de un string a mayúsculas, utilizamos "toUpperCase()", este método no toma parámetros.
Replace()
Para reemplazar fragmentos de un string con otro, podemos utilizar el método "replace()", este toma dos parámetros. El primer parámetro que toma el el fragmento a reemplazar y el segundo es el string con el que será reemplazado el string indicado. Solo reemplaza la primera ocurrencia del string indicado.
ReplaceAll()
Posee el mismo funcionamiento que "replace()", con la diferencia de que en lugar de solo reemplazar la primera ocurrencia, las reemplaza todas.

Estos son solo algunos de los métodos y propiedades disponibles para strings, el resultado que estos métodos devuelven solo son una copia de original con la modificación aplicada, es decir, el string original no es modificado en ningún momento.

Posición de un carácter

Es posible obtener el carácter en un punto específico de un string usando la notación "texto[]" dentro de estos corchetes se inserta el "índice" del carácter que queremos obtener (al igual que con los arrays, el índice comienza desde 0).
