Arrays

Los arrays son una manera especial de guardar listas de información, podría entenderse como una forma de envolver multiples valores en una sola variable.

Usualmente descrito como "list-like objects", son objetos que contienen multiples valores guardados como una "lista". Nosotros tenemos acceso a cada unos de estos elementos y podemos realizar acciones con cada uno de ellos de forma individual. Generalmente se utilizan para "envolver" valores que tienen algún tipo de relación. Por ejemplo, especies de animales, precios, artículos de una tienda, libro, etc. Sin los arrays. Sin embargo, los arrays no necesariamente tienen que ser contenedores de información relacionada.

Creando un array

Un array consiste de elemento separados por comas (",") envueltos entre corchetes ("[]").

En JavaScript, los arrays pueden contener prácticamente cualquier tipo de dato, objetos, otros arreglos e incluso combinaciones de estos.

El tamaño de un array

Para poder saber el tamaño de un array, es decir, la cantidad de elementos que posee, podemos usar el método "length", este nos devolverá la cantidad de elementos.

Acceso y modificación

Para conocer el indice de cualquier elemento tenemos el método "indexOf()" que toma como parámetro el valor del que queremos conocer su indice. Cuando el elemento se encuentra en el array, este método devolverá su indice, pero si este no se encuentra en el array, devolverá "-1".

Agregando y eliminando elementos

Existen métodos cuya función en especial es el agregar y eliminar elementos en un array, estos son:

Push()
El método "push()" sirve para agregar elementos al final de un array. Este toma como parámetro el elemento o elementos a agregar. Este método devuelve el nuevo tamaño del array.
Unshift()
El método "unshift()" sirve para agregar elementos al inicio de un array. Este toma como parámetro el elemento o elementos a agregar. Este método devuelve el nuevo tamaño del array,
Pop()
El método "pop()" sirve para eliminar el ultimo elemento en un array. Este devuelve como resultado el elemento eliminado.
Shift()
El método "shift()" sirve para eliminar el primer elemento en un array. Este devuelve como resultado el elemento eliminado.
Splice()
El método "splice()" sirve para poder eliminar un elemento en una posición dada y con la posibilidad de eliminar más elementos a partir de esta posición. "splice()" toma 2 argumentos, el primero para indicar el indice del elemento que queremos eliminar y le segundo para la cantidad de elementos a partir de este punto que queremos eliminar. Este método devuelvo los elementos eliminados.

Algo a considerar es que a diferencia de los métodos para strings, los métodos mencionados para arrays sí modifican el array original.

Acceso los elementos de un array

Para poder acceder a los elementos de un array de forma ordenada tenemos tres formas principalmente, para la mayoría de casos. Estos son:

Bucle for...of
Un bucle "for...of" sirve para recorrer un arreglo de forma ordenada de inicio a fin. En la mayoría de casos es la opción más versátil.
Map()
Algunas veces necesitaremos realizar una misma acción en cada elemento de un array y creando un nuevo array con los elementos modificados, para esta acción en particular existe el método "map()" el cual toma como argumento una función con las acciones que queremos realizar en cada elemento del array y devuelve como resultado una copia del array modificado.
Filter()
Algunas veces necesitaremos filtrar los elementos de un array y crear uno nuevo con los elementos filtrados, para esta acción en particular existe el método "filter()", el cual como parámetro toma una función "test". Cada elemento del array pasa por este "test" y si devuelven "true" como resultado, el valor se guarda en el array filtrado, si devuelven "false", se descarta el elemento y se pasa al siguiente.

Conversiones entre strings y arrays

Para esto tenemos 2 métodos principalmente:

Split()
Una forma de poder convertir una cadena de texto en un array es con el método "split()", en su forma más simple este toma un solo parámetro, un carácter o caracteres que fungirán como el separador para los diferentes elementos para el array.
Join()
Una forma de poder convertir un array en una cadena de texto es con el método "join()", este toma como parámetro el carácter o caracteres que separarán cada elemento del array al convertirlo a un string, puede dejarse en blanco. Otra forma de realizar esta misma acción es por medio del método "toString()", hasta cierto punto más fácil ya que no requiere de un parámetro, pero este siempre separa los diferentes elementos del array por medio de comas ",", a diferencia de "join()" donde se puede modificar este separador.
