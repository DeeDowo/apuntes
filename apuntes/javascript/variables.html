Variables

Una variable es un contenedor de un valor. Por ejemplo, una variable puede contener un numero (3) o un string ("texto") por mencionar algunos. En el caso de JavaScript, las variables se suelen usar especialmente para recordar cualquier tipo de información del usuario o para guardar referencias a elementos HTML. Definiendo una variable

En java script, se utiliza principalmente tres palabras claves para definir variables:

Const
La palabra clave "const" se utiliza para definir una variable la cual no cambiará su valor.
Let
La palabra clave "let" se utiliza para definir una variable la cual puede llegar a cambiar su valor.
Var
La palabra clave "var" al igual que "let", permite definir una variable a la cual se le puede llegar a cambiar el valor. Este era el estándar anteriormente para definir una variable. Sin embargo, actualmente su uso está desaconsejado debido a ciertos comportamientos conflictivos. Algunos de estos son:
  • "var" tiene un scope global, es decir, a diferencia de una variable cuyo scope suele ser el bloque en donde se declara, este no es el caso con var.
  • "var" permite la redefinición. Cuando JavaScript detecta que estamos tratando de volver a definir una variable preexistente, este nos lo impedirá, con "var" esto no sucede.
  • "var" añade propiedades al objeto global (window).

Cuando una variable está "definida", pero no inicializada, es decir, no tiene ningún valor asignado, esta devuelve "undefined". Para crear una variable solo hay que definirla, no es necesario inicializarla. También es posible inicializar un variable con "null", esto significa que esta está vacía de forma explicita.

La diferencia clave entre una variable sin inicializar y una variable no definida, es que la no inicializada es como una "caja" vacía, mientras que la variable sin definir, directamente no existe.

Constantes en JavaScript

Una constante es una variable cuyo valor asignado no puede modificarse, este puede ser una valor fijo o más comúnmente referencias a otros elementos, este comportamiento es similar al de un puntero constante en C++, es decir, esta variable siempre apunta a un mismo elemento, sin embargo este elemento puede cambiar, esto es notorio al asignar un objeto por ejemplo, la constante no podrá dejar de apuntar a este objeto, pero este objeto si puede cambiar características de este mismo.

Const o Let

El usar "const" o "let" al definir una variable depende del contexto, generalmente en un lenguaje uno usa const para cosas que no quiere que se modifiquen, mientras que let sirve para elementos que requieren un cambio, por ejemplo, contadores. Sin embargo en JavaScript en particular, la practica más común es utilizar "const" cada vez que sea posible y usar let solo cuando sea necesario.

Inicializando una variable

El inicializar una variable significa darle un valor a esta, esto se realiza utilizando la definición de la variable en conjunto con el operador "=" y el valor que queremos asignar a esta variable, una variable no necesariamente tiene que ser inicializada a la vez que es definida.

Actualizando una variable

Una vez una variable ha sido inicializada con un valor, este puede ser modificado siempre y cuando la variable no sea de tipo constante.

Nombrando variables

Una variable en JavaScript puede ser nombrada prácticamente de cualquier forma, sin embargo, existen algunas reglas:, uno debería usar solo nombres que contengan a-z, A-Z, 0-9 y el "_" (guión bajo)

  • Los nombres solo deben contener los caracteres a-z, A-Z, 0-9 y el "_" (guión bajo).
  • No usar "_" como inicio de una variable, estos tienen otro propósito en JavaScript cuando se usan al inicio.
  • No usar números al inicio de una variable.
  • Usar "lower camel case", es decir, usar minúsculas y solo usar una mayúscula al separar una palabra de otra.
  • Usar nombres intuitivos que puedan ser autodescriptivos sobre su contenido.
  • No usar palabras reservadas, como "var", "funtion", "let", etc.

Tipos de variables

En JavaScript existen diferentes posibles tipos de variables que podemos definir, algunos ejemplos son:

Numbers
Se pueden guardar números en variable, ya sean enteros (integers) o decimales (floats). A diferencia de otros lenguajes de programación, en JavaScript no es necesario indicar el tipo de dato de una variable. en el caso de los números, los valores no se asignan literalmente.
Strings
Los "strings" son cadenas de texto, estos deben asignarse utilizando "" (comillas doble) o ''(comillas simples) o ``(backticks).
Booleans
Los "booleans" son los valores "true" o "false" (verdadero o falso), si bien se puede asignar la palabra clave directamente, es más común usarlos para almacenar el resultado de comparaciones.
Arrays
Un array es un objeto único que contiene un conjunto de elementos, cada uno de estos elementos posee un lugar especificado por un indice, este comienza desde "0".
Objects
Los objetos son estructuras de datos que representan características de algo.

JavaScript es un lenguaje de tipado dinámico, esto significa que no es necesario especificar el tipo de información que se ingresará a una variable.
