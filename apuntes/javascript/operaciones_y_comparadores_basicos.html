Operaciones y comparadores básicos

Las matemáticas son parte fundamental de cualquier lenguaje de programación y JavaScript no es la excepción. A diferencia de otros lenguajes de programación, en JavaScript los números enteros (integers) y los numero de punto flotante (floats), entrar en el mismo tipo de dato y con estos se pueden realizar multiples operaciones. Por ejemplo:

+
Operador de suma, suma 2 valores, estos pueden ser literales o variables que los contengan.
-
Operador de resta, resta 2 valores, estos pueden ser literales o variables que los contengan.
*
Operador de multiplicación, multiplica 2 valores, estos pueden ser literales o variables que los contengan.
/
Operador de division, divide 2 valores, estos pueden ser literales o variables que los contengan.
%
Operador de residuo o modulo, es el sobrante entero al dividir 2 valores, estos pueden ser literales o variables que los contengan.
**
Operador de exponente, eleva un valor literal o variable a una cierta potencia.

Incremento y decremento

Cuando queremos incrementar o decrementar algún valor en JavaScript tenemos un par de operadores, estos son:

++i
El prefijo "++" indica que una variable se incrementará en 1 antes de la declaración.
i++
el sufijo "++" indica que una variable se incrementará en 1 después de la declaración.
--i
el prefijo "--" indica que una variable se decrementará en 1 antes de la declaración.
i--
el sufijo "--" indica que una variable se decrementará en 1 despues de la declaración.

Operadores de asignación

En JavaScript tenemos "shorthands" para realizar una operación en una variable y reasignar el valor resultante a esta misma. Algunos ejemplos son:

+=
Se suma la variable original con algo y el resultado se reasigna es esta.
-=
Se resta la variable original con algo y el resultado se reasigna en esta.
*=
Se multiplica la variable original con algo y el resultado se reasigna en esta.
/=
Se divide la variable con algo y el resultado se reasigna en esta.

Operadores de comparación

Los diferentes comparadores devuelven como resultado un valor booleano, es decir, "true" o "false" dependiendo de si la comparación es verdadera o falsa. Estos son:

===
compara 2 valores para ver si son iguales
!==
compara 2 valores para ver si son diferentes
<
compara 2 valores (menor que)
>
compara 2 valores (mayor que)
<=
comprara 2 valores (menor igual que)
>=
compara 2 valores (mayor igual que)
