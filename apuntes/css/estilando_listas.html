Estilando listas

las listas se pueden estilar como cualquier otro elemento en CSS, sin embargo, existen ciertas propiedades exclusivas para estas al igual que ciertas practicas que facilitan este proceso.

Si revisamos una lista sin ninguna modificación por nuestra parte usando las "dev tools" de un navegador, podremos ver como estas tienen estilos predefinidos al igual que varios elementos en CSS, una buena practica es "normalizar" estos.

Espacio en listas

Al estilar una lista, se tiene que poder mantener un orden y una consistencia, es decir, es preferente que nada esté "amontonado" o muy disperso. Una de las primeras propiedades que se pueden modificar para que una lista en una pagina se vea "homogénea" es el "line-height", de está forma todas las listas se verán con la misma separación que el resto de textos. Al hacer esto nos aseguramos de tener un mismo "ritmo vertical", el "line-height" más común es "1.5".

Estilos específicos para listas

Existen algunas propiedades especificas para listas, 3 principalmente para listas ordenadas y listas desordenadas:

  • list-style-type (fija el tipo de "bullets" que se usarán en una lista. Por ejemplo, círculos o cuadrados para una lista desordenada o números o letras para un lista ordenada)

  • list-style-position (fija si los "bullets" aparecen dentro o fuera del elemento "list item")
  • list-style-image (permite el uso de alguna imagen en particular como "bullets")
Bullet styles
la propiedad "list-style-type" permite modificar el tipo de "bullets" que una lista usará. Algunos ejemplo son:
  • list-style-type: upper-roman; (números romanos)
  • list-style-type: circle; (círculos vacíos)
  • list-style-type: square; (cuadrados)
  • list-style-type: none; (no habrán "bullets")
  • list-style-type: upper-alpha; (letras mayúsculas)
Bullet position
la propiedad "list-style-position" fija si las "bullets" aparecerán fuera o dentro (outside, inside) del elemento "list item", por defecto, se ubican fuera del elemento.
Custom bullet image
la propiedad "list-style-image" permite usar usa imagen personalizada como "bullets", sin embargo, es una propiedad un tanto limitado en términos de controlar la posición, tamaño, etc. Una mejor alternativa es usar las propiedades para "backgrounds".

List style shorthand

Las propiedades mencionadas anteriormente "list-style-type", "list-style-position" y "list-style-image" pueden ser fijadas usando la propiedad shorthand "list-style". Los valores pueden ser fijados en cualquier orden. Cuando se agrega tanto un tipo de "bullet" como una imagen personalizada, el tipo funciona como un "fallback" para cuando la imagen no logré ser aplicada correctamente.

Conteo en una ordered list

En el caso de las "ordered list" (ol) a veces uno puede querer contar en un orden diferente al predeterminado, ya sea empezando desde un numero diferente de 1 o contar en orden descendente o contando en pasos de más de uno (1-3-6-9...), etc. Para esto existen tanto propiedades en CSS como atributos en HTML:

Start
El atributo HTML "start" permite fijar desde que número se empezará el conteo en una "ordered list"
Reversed
El atributo HTML "reversed" permite contar en sentido contrario. Se le puede asignar como valor desde que número comenzara el retroceso, de lo contrario empezará desde el 1 (1, 0, -1, -2, ...).
Values
El atributo HTML "values" permite fijar explícitamente el valor de un elemento en una "ordered list"

Incluso si decidimos que no se muestren visualmente estos números, por semántica, siempre deben estar presentes en HTML

Background para "bullets"

  1. Por defecto una un lista tiene 40px de "margin-left" antes de todos sus "list items", podemos quitarle 20px a este, luego agregar 20px como "padding-left" en el propio "list item" y de esta forma seguirán alineados, pero ahora tendremos el espacio disponible dentro de cada uno para poder agregar las imágenes personalizadas.
  2. Quitamos los "bullets" predeterminados con "list-style-type: none".
  3. Agregamos la imagen con "background-image" y ajustamos la "position" a 0 0, de está forma siempre aparecerá la imagen en la esquina superior izquierda que es donde la queremos.
  4. Personalizamos el tamaño de la imagen usando "background-size", de preferencia esto debería ser una unidad relativa y ser un poco mayor o menor que le texto mismo.
  5. Por ultimo, aplicamos "background-repeat: no-repeat;" para que el "bullet" solo aparezca una vez.

Con este método podemos agregar cualquier imagen como un "bullet" personalizado, también es posible agregarlos por medio de "::before".
