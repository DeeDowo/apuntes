Estilos para textos

Uno de los usos principales para CSS es el de estilizar textos. El texto dentro de un elemento se acomoda en la parte del "contenido" de una "caja", el cómo se acomoda un texto depende del idioma, pero generalmente este comienza a ordenarse desde la parte superior izquierda y continúa hasta el final de la línea disponible, una vez lo alcanza, continúa en la siguiente línea y repite este proceso una y otra vez.

Las propiedades para comenzar a estilizar textos pueden separarse en dos categorías principalmente:

Font styles
Propiedades que afectan el cómo se verá un texto.
Text layout
Propiedades que afectan el cómo se distribuirá un texto.

Sin importar la clase de modificaciones que queramos realizar en un texto, estas se aplicarán a todo el texto de dicho elemento y estas se heredarán a sus hijos posteriores sin importar el nivel de anidación. Para poder afectar solo ciertos "trozos" en particular de un texto, se necesita el uso de otras etiquetas como <span> ... </span> o algún pseudo elemento como "::first-letter" (envuelve la primera letra de algo) o "::first-line" (envuelve la primera línea de algo).

Algunos conceptos para fuentes

Fuentes
Una "fuente" es el estilo tipográfico que tendrá un texto. En CSS tenemos dos propiedades para personalizar estas y sus características.
Web safe fonts
Existen fuentes que están disponibles en prácticamente todos los sistemas operativos, estas son llamadas "web safe fonts", algunos ejemplos son:
  • Arial o Helvetica
  • Courier New o Courier
  • Georgia
  • Times New Roman o Times
  • Trebuchet MS
  • Verdana
Estas son solo algunas de las "web safe fonts" disponibles. La principal función de estas es ser "fallbacks" para cuando pudiera ocurrir algún problema al intentar cargar nuestra o nuestras fuentes principales.
Tipos de fuentes
Generalmente las fuentes llevan en sus nombres el tipo de fuente que son, algunos ejemplos serían:
  • serif (texto con decoraciones. generalmente para títulos)
  • sans-serif (sin serif, es decir, textos sin decoraciones. generalmente para texto normal)
  • monospace (texto donde cada carácter tiene el mismo tamaño. generalmente para programación o para caracteres en diferentes idiomas)
  • cursive (emula la cursiva)
  • fantasy (decorativas. no es texto en sí)
El agregar estos tipos como fallbacks, hará que se aplique la primera fuente que sea del tipo especificado.
Font stacks
Es el conjunto de múltiples fuentes que especificamos para que el navegador tenga opciones en caso de que una no se encuentre disponible, cada fuente se separa por comas y van en orden de aplicación, es decir, primero se agrega el "caso ideal", luego algunas opciones y por último los tipos más genéricos posibles, pero que hasta cierto punto sigan funcionando para nuestro diseño. Por ejemplo:
  • font-family: "fuente deseada", arial, sans-serif;
Algo a considerar es que aquellas fuentes cuyos nombres tengan espacios de por medio, es recomendable ponerlas entre comillas al igual que cualquier nombre que pueda ser interpretado como un tipo genérico o una palabra clave en CSS.

Algunas propiedades para el "Font style"

Color
"color" en el caso de textos, cambia su color, pero en realidad lo que hace es fijar el color del "foreground", es decir, lo que está al frente de los elementos como el underline, el overline o más comúnmente las fuentes.
Font family
"font-family" aplica una fuente o lista de fuentes diferentes al texto. El navegador solo aplicará una fuente, en el caso de una lista de fuentes, la primera disponible, si ninguna está disponible el navegador usa la fuente por defecto de este.
Font size
"font-size" sirve para definir el tamaño que tendrá una fuente dentro de un elemento. Los valores más comunes que se usan para esto son:
  • px (píxeles)
  • em (unidad relativa al tamaño del texto del padre)
  • rem (unidad relativa al tamaño del texto del elemento raíz)
Por defecto el "font-size" es heredado del elemento padre, esto comienza desde el elemento raíz, cuyo estándar es de 16 px.
Font style
"font-style" se usa para poner un texto en modo "italic" (simular cursiva), recibe 3 valores:
  • normal (sin itálica)
  • italic (muestra el texto en su forma itálica si es que esta existe)
  • oblique (simula una forma itálica si esta no existe por defecto)
Cuando se usa "italic" y no existe esta versión de la fuente, se aplica "oblique" internamente
Font weight
"font-weight" se usa para especificar el grosor de un texto. Por lo general cada fuente puede tener su propio rango de grosores (light, normal, bold, extrabold, black, etc.), pero en la realidad solo suelen usarse 2 tipos comúnmente:
  • normal (grosor normal)
  • bold (grosor mayor al normal)
Otras opciones comunes son "lighter" y "bolder", las cuales fijan un grosor menor o mayor respecto al grosor de light y bold. El grosor también se puede expresar numéricamente lo cual da mayor control sobre este en comparación con el uso de palabras clave, el rango generalmente es de 100-900.
Text transform
"text-transform" como su propio nombre lo indica, permite la transformación de un texto, puede tomar 5 valores:
  • none (previene cualquier transformación)
  • uppercase (transforma todo el texto a mayúsculas)
  • lowercase (transforma todo el texto a minúsculas)
  • capitalize (transforma todas las iniciales de cada palabra en mayúsculas)
  • full-width (normaliza diferentes escrituras o símbolos)
"text-transform: full-width;" en especial, funciona como si creara una cuadrícula donde cada carácter o símbolo abarcará todo el espacio disponible en su respectivo contenedor, esto hace que todos los símbolos o caracteres posean el mismo tamaño y distribución lo cual hace que tengan un estilo visual consistente. Generalmente se usa para caracteres o símbolos en otro idioma.
Text decoration
"text-decoration" se usa para activar o desactivar decoraciones en textos, un ejemplo, el subrayado. Los valores que puede tomar principalmente son:
  • none (desactiva todas las decoraciones presentes en un texto)
  • underline (rayado debajo del texto)
  • overline (rayado sobre el texto)
  • line-through (rayado en medio del texto, lo "tacha")
A su vez, esta propiedad es un shorthand para "text-decoration-line", "text-decoration-style", "text-decoration-color". Por ejemplo:
  • text-decoration: line-through red wavy;
Text shadow
Se pueden agregar sombras a un texto usando la propiedad "text-shadow", puede tomar 4 valores:
  • A qué distancia aparecerá la sombra horizontalmente
  • A qué distancia aparecerá la sombra verticalmente
  • El blur, es decir, qué tan sólida se verá la sombra. Cuanto más alto el valor, menos sólida y clara será la sombra. Su valor es 0 por defecto
  • El color, el valor por defecto es el propio color actual
Se pueden agregar múltiples sombras separando los valores de cada una por una ','.

Algunas propiedades para el "Text layout"

Una vez entendidas algunas de las propiedades esenciales para la edición de textos, el siguiente paso es el "layout", es decir, de qué forma se distribuyen estos.

Text align
La propiedad "text-align" se usa para controlar la alineación de un texto, es decir, desde donde fluye el texto. Existen cuatro valores:
  • center (centrado)
  • left (izquierda)
  • right (derecha)
  • justify (justificado)
El único valor con el que realmente hay que tener "cuidado" es con el texto justificado ya que este hará que el texto tenga espacios diferentes, esto para que cada línea cubra exactamente todo el espacio disponible. Generalmente se ve mal.
Line height
La propiedad "line-height" dicta la altura de cada línea de texto (line-box). La mejor práctica para asignar es usar valores sin unidades, es decir, multiplicadores. Una altura recomendada es 1.5 o 2 (doble espacio) estos significan 1.5 y 2 veces el tamaño del texto.
Letter spacing y Word spacing
Las propiedades "letter-spacing" y "word-spacing" son similares a "line-height", estas sirven para modificar el espaciado entre letras y palabras respectivamente. Al igual que con "line-height", la mejor práctica al asignar un valor es usando multiplicadores.

Font shorthand

Varias de las propiedades descritas pueden ser modificadas mediante el shorthand "font", estas son las siguientes:

  • font-style
  • font-variant
  • font-weight
  • font-stretch
  • font-size
  • line-height
  • font-family

De entre todas las propiedades a asignar disponibles, las únicas requeridas tanto en presencia como en orden son "font-size" y "font-family", es decir, el "font-size" siempre debe declararse antes de "font-family"

Otra regla para este shorthand es que el "font-size" siempre debe ir separado del "line-height" por medio de un '/'.

Existen más propiedades disponibles para modificar textos, tanto para los estilos como para la distribución.
