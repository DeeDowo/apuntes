Valores en CSS

Cada propiedad en CSS tiene uno o varios "tipos de valores", estos definen qué valores acepta una propiedad.

Qué es un valor CSS?

Un valor es una forma de expresar algo, por ejemplo un color usando la función rgb o una dimensión usando píxeles.

Qué es un tipo de valor?

Un tipo de valor es el conjunto de las diferentes formas de expresar un valor. Por ejemplo, cuando decimos que una propiedad acepta como tipo de valor un color, nos referimos a que acepta cualquiera de las formas para expresar un color.

Números

Existen varias formas para expresar números:

  • integer (números enteros, negativos o positivos)
  • number (números decimales, puede o no tener el punto decimal, tanto negativos como positivos)
  • dimension (un número con una unidad)
  • percentage (un porcentaje de otro elemento)

Unidades

Uno de los tipos de datos más comunes en CSS, existen dos tipos:

Absolutas
Tienen un tamaño predefinido que no cambia (cm, mm, q, in, pc, pt, px). El uso principal de las unidades absolutas es para imprimir contenido más que para mostrarlo.
Relativas
Dependen de otro valor de referencia para definir su propio tamaño, pueden ser relativas al viewport o relativas al contexto.
Las unidades relativas al viewport se basan en el tamaño de la ventana del navegador (viewport).
  • vw (1% del ancho del viewport, útil para diseños responsivos)
  • vh (1% del alto del viewport, útil para secciones a pantalla completa)
  • vmin (1% del lado más pequeño del viewport)
  • vmax (1% del lado más grande del viewport)
Las unidades relativas al contexto dependen de valores como el tamaño de la fuente o dimensiones de elementos "padres".
  • em (relativa al "font-size" del elemento actual. Cuando se usa para definir el "font-size" la referencia es el "font-size" del elemento padre)
  • rem (relativa al font-size del elemento raíz)
  • % (relativa al tamaño del elemento padre, su principal uso es para el width/height)
La principal ventaja del uso de las unidades relativas es la flexibilidad que ofrecen al no ser valores fijos.

Color

Los colores son valores que pueden ser utilizados en una gran variedad de propiedades en CSS, ya sea para el color de un texto, bordes, fondos y más. El estándar actualmente permite soportar colores en 24-bits, es decir, mostrar aproximadamente 16.7 millones de colores distintos, mediante la combinación de rojo, verde y azul con 256 diferentes valores cada uno (256 x 256 x 256 = 16,777,216). Algunas formas de expresar colores son:

Color keywords
La forma más básica de expresar un color es mediante una palabra clave (un nombre), son fáciles de usar y entender, pero limitados en la cantidad de colores que se pueden expresar.
Color hexadecimal RGB values
Expresa un color en formato hexadecimal con el prefijo '#', se puede usar con 3 o 6 caracteres en hexadecimal (#123 o #123456), con la opción de agregar uno o dos caracteres más para representar el canal alpha (opacidad).
Color RGB values
Usando la función "rgb()", toma 3 parámetros (0-255 o 0%-100%) separados por espacios que representan el rojo, verde y azul con la opción de un cuarto valor separado por un '/' para representar el canal alpha (opacidad) (0-1).
Uso de Tonalidades (hue)
Además de palabras clave, hexadecimales o la función "rgb()", podemos especificar un color mediante el uso de tonalidades (hues) basadas en el círculo cromático y sobre estas aplicar un mayor o menor nivel de brillo o saturación, de esta forma podemos trabajar de forma más "intuitiva" sobre un mismo color.
El "hue" representa el tono base de 0 a 360, haciendo referencia a un "círculo", donde:
  • 0 o 360 = rojo
  • 60 = amarillo
  • 120 = verde
  • 240 = azul
  • 300 = magenta
Una vez fijado el tono base, tenemos 2 funciones principalmente:
  • hwb( hue(tono) whiteness(blancura 0%-100%) blackness(negrura 0%-100%) / alpha(opacidad 0-1 o 0%-100%) )
  • hsl( hue(tono) saturation(saturación 0%-100%) lightness(luminosidad 0%-100%) / alpha(opacidad 0-1 o 0%-100%) )

Imágenes

Este tipo de valor hace referencia a utilizar una imagen apuntada por medio de la función url o a un degradado

Posición

El tipo de valor "position" representa coordenadas 2D (x, y), usadas para posicionar un elemento, por ejemplo una imagen. Puede tomar keywords (top, left, bottom, right y center) o longitudes, las cuales representan distancias a partir del lado superior y el lado izquierdo de la "caja".

Usualmente consiste de 2 valores, el primero controla el eje horizontal (x) y el segundo controla el eje vertical (y), cuando solo uno de estos se especifica, al restante por defecto se le asigna un valor de "center".

Strings

El tipo de valor "string", sirve para ingresar textos entre comillas, por ejemplo, con la propiedad "content".

Funciones

Similar a cualquier lenguaje de programación, las funciones sirven para realizar "algo" con ciertos parámetros. La principal ventaja de usar funciones es que hacen más flexible nuestro código. Algunas funciones comunes son:

  • calc()
  • translate()
  • scale()
  • rotate()
  • min()
  • max()

Existen muchos más tipos de valores y formas de expresar dichos valores en CSS, pero estos pueden considerarse como los más comunes.
