Estilando links

Los links tanto como sus estilos son una parte fundamental de la gran mayoría de páginas, tanto estáticas como dinámicas.

Estados de un link

La primera cosa que hay que entender sobre links es el concepto de los diferentes estados de un link, los cuales pueden ser estilados usando diferentes pseudo-clases:

Link
La pseudoclase ":link", para links aún no visitados.
Visited
La pseudoclase ":visited", para links ya visitados (existen en el historial del navegador).
Active
La pseudoclase ":active", para links que están siendo presionados.

Estilos por defecto

Los links tienen estilos predeterminados, estos son:

  • Underline
  • Links sin visitar son azules
  • Links visitados son púrpuras
  • Al apuntar un link con el mouse cambia de una flecha a una mano
  • Links activos cambian a rojo

La mayoría de estos estilos son iguales a los estilos por defecto de 1990s, esto es porque los usuarios conocen y esperan estos comportamientos en cualquier link o al menos algo similar. Al nosotros querer estilizar links, estos tienen que al menos crear una nueva métrica homogénea e intuitiva. Un link debe poder entenderse como tal de forma intuitiva.

Iconos para links

Una práctica común al estilar links es el agregar iconos a estos para de esta forma crear un indicador visual sobre qué clase de link se trata. Por ejemplo, se puede agregar un pequeño icono que haga referencia a que se redireccionará a otra página. Los iconos se suelen agregar usando los pseudoselectores ::after o ::before.

  1. Primero usamos un selector para un elemento "a" con un atributo "href" y el cual tiene que tener como inicio "http" en su valor, esto indicaría un link a una nueva página, todo en conjunto con el pseudoselector "::after" para agregar contenido al final del elemento seleccionado.
  2. Una vez editando las reglas de nuestro selector, usamos "content" con un string vacío (""), lo cual crea una "caja" vacía al final del link donde podremos insertar el icono.
  3. Asignamos un display "inline-block"
  4. Asignamos un tamaño para esta caja y un margen para que no esté "pegado" al link.
  5. Por último agregamos una imagen con un "background-size: 100%;" lo cual hará que este icono abarque todo el espacio posible en su contenedor, pero sin afectar su aspect ratio.

Con este método podemos hacer aún más fácil el entendimiento de nuestros links.

Estilando links como botones

Actualmente por lo general los links se suelen estilar más cómo botones. Por ejemplo, en el caso de una barra de navegación. Una forma sencilla de hacer esto es agregando un color de fondo, padding, centrando el texto y un borde redondeado.
