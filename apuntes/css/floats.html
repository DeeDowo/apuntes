Elementos flotantes

La propiedad "float", creada originalmente para imágenes "flotantes" dentro de un bloques de texto. Fue introducida para permitir agregar imágenes dentro de una columna de texto, donde el texto envolvería a la imagen a la derecha o izquierda de este, similar a un los artículos en un periódico o revista. Sin embargo, esto no estaba limitado a imágenes, podía realizar dicha función con cualquier elemento, entonces se popularizó el efecto de "drop-caps", actualmente dicho efecto se puede realiza con la propiedad "initial-letter" de una manera mucho más sencilla.

Los "floats" solían usarse incluso para crear layouts completos de una pagina, pero actualmente no deberían usarse para más que su propósito inicial, ya que ahora existen multiples técnicas más actuales tanto para layouts como para efectos para los que solía servir "float".

Uso de float

La propiedad "float" hace que un elemento salga parcialmente del flujo normal "pegándose" a un lado de su contenedor (izquierda/derecha). su efecto sobre otros elementos depende del tipo de elemento, es decir:

Elemento block
Estos elementos ignoran al elemento flotante y toman su espacio como si no existiera ahí.
Line boxes
Se ajustan su flujo para rodear al elemento flotante.

Uso de clear

La propiedad "clear" se relaciona con "float" ya que hacen que un elemento "ignore" a estos elementos flotantes, es decir, respetará el espacio como si el elemento no fuera de tipo flotante.

Left
"clear: left;" solo ignora elementos flotantes a la izquierda.
Right
"clear: right;" solo ignora elementos flotantes a la derecha.
Both
"clear: both;" ignora elementos flotantes a ambos lados.

Una forma para aislar elementos flotantes con solo ciertos elementos, es decir, que solo algunos elementos de la pagina reconozcan y "envuelvan" al elemento flotante, se logra agrupándolos en algún contenedor común para estos y dándole la propiedad a este contenedor de "display: flow-root", de esta forma solo los elementos relacionados dentro del del contenedor envolverán al elemento flotante, mientras que el resto de elementos en la pagina solo lo ignorarán y respetarán su espacio original.
