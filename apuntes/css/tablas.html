Estilando tablas

Al ver una tabla por primera vez, podremos notar como esta se ve "amontonada", esto se debe a que en las tablas, cada celda solo toma el espacio necesario por defecto. Estos estilos predeterminados son malos objetivamente y siempre deberían modificarse al menos un poco para tener así una mayor claridad. El estilar una tabla es un trabajo relativamente sencillo, siempre y cuando esta se haya creado utilizando los elementos disponibles en HTML de forma correcta. Algunas propiedades especialmente útiles son:

"table-layout:fixed"
Esta propiedad permite fijar tamaños específicos para las celdas, de esta forma podremos crear una tabla menos "amontonada".
"border-collapse:collapse"
Esta propiedad hace que los diferentes bordes de una tabla se "junten". Por defecto los bordes de una tabla se muestran de forma individual (cada celda y la propia tabla tiene su borde), al usar "border-collapse:collapse" los diferentes bordes se combinan dando como resultado que solo se muestre un borde en general.

Lineas de zebra

"Las lineas de zebra" son un efecto común, pero muy util para mejorar la legibilidad de una tabla. Este efecto, como su nombre da a entender, significa que el color de las diferentes filas cambia simulando el patrón de las franjas de una zebra.

En el código anterior se aplica un "background-color" a las filas de una tabla (anidadas en el tbody explicito o implícito) con ayuda de las pseudoclase "nth-child", cuyo propósito es apuntar a un hijo en particular, en este caso lo usamos en conjunto con una función para aplicar una regla a los hijos pares y una a los hijos impares y de esta forma es como podemos crear de forma sencilla el patrón de "lineas de zebra" en nuestras tablas.

La posición del "caption" de una tabla también se puedo modificar. Con la propiedad "caption-side" podemos elegir donde se mostrará (top/bottom).
